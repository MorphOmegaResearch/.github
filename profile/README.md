# Morph-Omega Research

Novel Neural architecture research. New Zealand. 2025–2026.

---

## What This Is

Two systems built sequentially — the first parenting the second.

**System 1 — LLM Learning Engine** (2025 → present)
A multi-tab Python GUI for model training, pattern exploration, and inference evaluation.
Built to test hypotheses about semantic states, bi-hemispheric inference, and on-sight training.
Every core property of Morph-Omega was confirmed here before the substrate existed.

**System 2 — Morph-Omega** (2025 experimental → 2026 formalised)
A neural architecture in which the pattern database, vocabulary, and weight matrix are unified
as a single canonical substrate. Each pattern carries a Unique identifier that simultaneously
serves as its token ID and its index into the embedding weight matrix.

> *"pattern = vocabulary = weight"*

## The Origin

The architecture was not invented theoretically. It was distilled from a working semantic
state system (ZPR / CTP / Delta) that preceded it and confirmed the core mechanics before
formalisation. The SR output format instilled via system prompt in 2025 is the same
measurement schema the substrate computes natively in 2026.

Experimental work: ~August 2025. Formal architecture registration: January 2026.
Proof of concept substrate: 265,598 patterns on commodity laptop hardware.
